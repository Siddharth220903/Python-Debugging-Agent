# RAG Documentation

This folder contains the use of LangChain APIs to create a knowledge set for debugging agent. It is assumed that passing the appropriate content from documentation is more relevant than passing the error encountered while running the file.

## Currently Supported Sources:
1. Python 3.11 Documentation: [https://docs.python.org/3.11/](https://docs.python.org/3.11/): 
    Downloaded text files from https://docs.python.org/3.11/download.html which are closest to natural language for context to LLM Model.
